E. coli Genome Annotation Project

Pipeline Steps:

1. Raw sequencing data downloaded from NCBI SRA
   Accession: SRR2584863

2. Quality assessment
   Tool: FastQC

3. Read trimming
   Tool: Trimmomatic

4. Genome assembly
   Tool: SPAdes

5. Assembly evaluation
   Tool: QUAST

6. Genome annotation
   Tool: Prokka
